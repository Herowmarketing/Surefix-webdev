# Spline 3D Component — Project Asset

## Overview
A reusable React component for embedding interactive Spline 3D scenes in Next.js / React projects.
Package: `@splinetool/react-spline` + `@splinetool/runtime`

## Installation
```bash
pnpm add @splinetool/react-spline @splinetool/runtime
# or
npm install @splinetool/react-spline @splinetool/runtime
```

## Component Code
```tsx
'use client'

import { Suspense, lazy } from 'react'
const Spline = lazy(() => import('@splinetool/react-spline'))

interface SplineSceneProps {
  scene: string
  className?: string
}

export function SplineScene({ scene, className }: SplineSceneProps) {
  return (
    <Suspense 
      fallback={
        <div className="w-full h-full flex items-center justify-center">
          <span className="loader"></span>
        </div>
      }
    >
      <Spline
        scene={scene}
        className={className}
      />
    </Suspense>
  )
}
```

## Notes
- Always use `lazy()` + `Suspense` to avoid SSR issues
- Spline scenes are hosted at `https://prod.spline.design/`
- Create scenes at https://spline.design/
- Best used for hero sections, product showcases, or interactive backgrounds
- Performance tip: Only load Spline when the section is in viewport using IntersectionObserver
